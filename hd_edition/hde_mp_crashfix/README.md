#### Description
Basic community patch for fixing a crash occuring during multiplayer games in 
Heroes of Might and Magic HD Edition. CRT not used, compiles to a ~5KB .exe

#### How to use
1. Start HD Edition
2. Start hde_mp_crashfix.exe
3. Play multiplayer
